# 🍽️ FoodExpress

A full-stack online food ordering web application developed using Java, JSP, Servlets, JDBC, and MySQL. The application allows users to browse restaurants, explore menus, place food orders, and manage their accounts through a responsive web interface.

---

## 🚀 Live Demo

🌐 https://foodexpress-uokd.onrender.com

---

## 📖 Project Overview

FoodExpress is an online food delivery platform where users can:

- Register and Login
- Browse Restaurants
- View Restaurant Menus
- Add Items to Cart
- Place Orders
- View Order History

Administrators can:

- Admin Login
- Dashboard
- Restaurant Management
- Menu Management
- Order Management
